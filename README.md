# AFF3CT-Lab

**AFF3CT-Lab** is a research-oriented extension of the AFF3CT library.

This repository is built on top of AFF3CT and is used to develop and evaluate **new code constructions and decoding algorithms** for error-correction codes.

---

## Purpose

* Implement custom decoders
* Explore new construction methods
* Prototype and test research ideas

---

## Current Work

* DSCF-based decoding improvements
* Metric-driven decoding strategies
* Perturbation Enhanced-SC and -SCL decoders

---

## Disclaimer

This is an **experimental research fork** of AFF3CT.
Features may be unstable and under active development.
